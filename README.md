# 创建项目
## npm命令
```npm install -g yo```

```npm install -g gulp```

```npm install -g bower```

```npm install -g generator-express```

## 使用技术
生成器：generator-express

项目：bower\gulp

前端：bower+bootstrap

后端：MVC+config

## 前台技术
bootstrap ```bower install --save bootstrap```

font-awesome ```bower install --save components-font-awesome```
